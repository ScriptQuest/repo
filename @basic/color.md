**Description:** Used to set the color of the text

## Depends on:
none

## Example usage:
```color rB "Hello World!"``` - Red text (bold) <br/>
```color r "Hello World!"``` - Red text <br/>
```color B "Hello World!"``` - Bold text <br/>
```color "Hello World!"``` - Default color (no changes)

## Available colors:
r - Red <br/>
g - Green <br/>
b - Blue <br/>
y - Yellow <br/>
p - Purple <br/>
c - Cyan

## Available styles:
B - Bold
